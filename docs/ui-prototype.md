# Прототип пользовательского интерфейса

Ниже — low-fidelity прототип ключевых экранов клиентского сценария.

## Экран 1: Каталог

```text
+--------------------------------------------------------------------------------+
| FastFood PreOrder                     [Войти] [Мои заказы]                     |
+--------------------------------------------------------------------------------+
| Категории: [Бургеры] [Напитки] [Закуски] [Десерты]                             |
+--------------------------------------------------------------------------------+
| Бургер Классический              249 ₽     [ - ] 1 [ + ]    [В корзину]       |
| Двойной Чизбургер                329 ₽     [ - ] 0 [ + ]    [В корзину]       |
| Картофель Фри                    139 ₽     [ - ] 2 [ + ]    [В корзину]       |
| Кола 0.5                         119 ₽     [ - ] 1 [ + ]    [В корзину]       |
+--------------------------------------------------------------------------------+
| Корзина: 3 поз. / 647 ₽                                      [Оформить заказ]  |
+--------------------------------------------------------------------------------+
```

## Экран 2: Оформление заказа

```text
+--------------------------------------------------------------------------------+
| Оформление заказа                                                               |
+--------------------------------------------------------------------------------+
| Телефон:              +7 (___) ___-__-__                                        |
| Точка самовывоза:     [ ТЦ Центральный ▼ ]                                      |
| Время получения:      [ 19:30 ▼ ]                                               |
| Комментарий:          [ Без лука ]                                              |
+--------------------------------------------------------------------------------+
| Состав заказа:                                                                  |
| - Бургер Классический x1 ............................................... 249 ₽ |
| - Картофель Фри x2 ..................................................... 278 ₽ |
| - Кола 0.5 x1 .......................................................... 119 ₽ |
+--------------------------------------------------------------------------------+
| Итого: 647 ₽                                             [Подтвердить заказ]    |
+--------------------------------------------------------------------------------+
```

## Экран 3: Статус заказа

```text
+--------------------------------------------------------------------------------+
| Заказ № A1024                                                                   |
+--------------------------------------------------------------------------------+
| Статус: [ NEW ] -> [ IN_PROGRESS ] -> [ READY ] -> [ CLOSED ]                  |
| Текущий статус: IN_PROGRESS                                                     |
| Самовывоз: ТЦ Центральный, 19:30                                                |
| Телефон: +7 (999) 000-00-00                                                     |
+--------------------------------------------------------------------------------+
| [Обновить]                                                       [Отменить]     |
+--------------------------------------------------------------------------------+
```

## Короткий пользовательский сценарий
1. Пользователь выбирает товары на экране каталога.
2. Переходит к оформлению и указывает контакт/точку/время.
3. Подтверждает заказ и отслеживает статус до готовности.
